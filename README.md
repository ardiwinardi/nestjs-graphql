# GraphQL API

This app demonstrates how to build a back-end app using Nest.js with GraphQL & Passport

## Features

- Authentication using passport
- Queries & Mutations of User
- Queries & Mutations of Product
- Queries & Mutations of Category

## Stacks

- Nest.js
- MongoDB
- TypeScript
- GraphQL
- Passport

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
